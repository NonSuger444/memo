# 作業メモ

1. 「Markdown Preview Enhanced」プラグインをアンインストール
2. 「Markdown Preview Mermaid Support」プラグインをインストール
3. setting.jsonに以下の設定を追加

```json
// Markdown PDFでMermaidを使用する際のCDN URLを指定
"markdown-pdf.mermaidServer": "https://cdn.jsdelivr.net/npm/mermaid@9.4.3/dist/mermaid.min.js",
```

4. keybindings.jsonに以下の設定を追加

```json
// プレビューをサイドに表示
{
    "key": "ctrl+shift+v",
    "command": "markdown.showPreviewToSide",
    "when": "editorTextFocus && editorLangId == markdown"
},
// Shift+Enterで改行
{
    "key": "shift+enter",
    "command": "type",
    "args": { "text": "  \n" },
    "when": "editorTextFocus && editorLangId == 'markdown'"
}
```

5. 「Markdown PDF」プラグインを入れてない場合は導入する
6. 「vscode-pdf」プラグインを入れる
